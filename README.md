## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/KHNRV/super_math_challenge.git
   ```
2. cd into the game's root folder

3. Launch the game
   ```sh
   ruby main.rb
   ```
4. Have fun!
